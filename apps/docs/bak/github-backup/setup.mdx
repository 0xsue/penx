# Setup GitHub backup

## 1. Get a GitHub account

Get a GitHub account in: [github.com/signup](https://github.com/signup) , if you have one, go to step 2.

## 2. Create a repository

Login to [github.com](https://github.com/) Create a new repository in [github.com/new](https://github.com/new) , checkout the guide image below:

![create-repo](/images/create-repo.png)

Tow important notes:

- select "private" (Create a private repository);
- Add a README file (It will create the initial commit);

## 3. Ouath GitHub account to PenX

In PenX, click the "Setup GitHub backup" button, you will see the Penx GitHub backup page.

In this page, click the "Connect GitHub" button and connect your GitHub account by Oauth2:

![](/images/github-oauth.png)

## 4. Install "PenX sync" App

if your GitHub account is connected to PenX, then you can install "PenX sync" GitHub app.

Add a GitHub account:
![](/images/add-github-account.png)

Select an GitHub account:
![](/images/select-github-acount.png)

Install the "PenX sync" GitHub app:
![](/images/install-penx-sync.png)

## 5. Connect Github repository

After installing the "PenX sync" GitHub app, click the "Connect" button to connect a GitHub repository:

![](/images/connect-repo.png)

if you connect a GitHub repository successfully, you will see this UI:
![](/images/github-connected.png)

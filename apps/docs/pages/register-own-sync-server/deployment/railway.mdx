# Deploy to Railway

## Prerequisites

Before deployment, Please read the [Prerequisites](/register-own-sync-server/deployment/prerequisites) carefully.

## On click deploy PenX sync server to Railway.

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/L-0Ee-?referralCode=tdmYvn)

## Real world example

When deployed successfully, you should get a URL like:

https://sync-server-dev.up.railway.app/

Now copy this server URL to PenX server settings.

## Upgrade

If new version PenX sync server is released, you can upgrade it easily by click the redeploy button in Railway dashboard.

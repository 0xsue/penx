# Register own sync server in 3 steps

## 1. Create a sync server in PenX

Login to [app.penx.io](https://app.penx.io), go to the **Sync server** Page, and create a sync server. You will get a sync server **Token**.

![sync-server-entry](/images/sync-server-entry.png)

![create-sync-server-modal](/images/create-sync-server-modal.png)

## 2. Deploy a sync server

Deploy a your own sync server, and get a sync server URL.

It can be deployed on your infrastructure and hosting platform of choice.

- [Fly.io](/register-own-sync-server/flyio)
- [Railway](/register-own-sync-server/railway)
- [Docker](/register-own-sync-server/docker)
- [Docker Compose](/register-own-sync-server/docker-compose)

## 3. Update Sync server URL in PenX

Back to [app.penx.io](https://app.penx.io), and update the PenX's sync server URL settings with the sync server URL from step 2.

![edit-sync-server-url](/images/edit-sync-server-url.png)
